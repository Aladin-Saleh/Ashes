Detailed Description

template<typename FlagType>
class ashes::FlagCombination< FlagType >

Template class providing implicit conversion from a scoped enum to a basic integer type.

Remarks
Allows definition of flags, through binary operations (&, |), for which operands will be of same binary size.

FlagTypeThe scoped enum type.
